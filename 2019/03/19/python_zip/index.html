1、zip基本说明

zip函数用于将可迭代的对象作为参数，将对象中对应的元素打包成一个个元组，返回由这些元组组成的列表。在Python3.x中为了减少内存，返回的是一个对象，可以用list()转换来输出列表。

如果各个迭代器的元素个数不一样，则返回列表长度与最短的对象相同，利用*号操作符，可以将元组解压为列表。
